# DME Synthesis Review

合成ガス（CO + H₂）からのDME（ジメチルエーテル）合成に関する技術総説と、主要プレイヤー・ライセンサーのプロセス比較をまとめた自己完結型HTML資料。

## 内容

| ファイル | 内容 |
|---|---|
| [`dme_synthesis_review.html`](dme_synthesis_review.html) | DME合成の技術総説（基礎反応・熱化学・2ルート比較）と各社プロセス比較表（JFE・KOGAS・Haldor Topsøe・Air Products 他） |
| [`dme_review_matrix.html`](dme_review_matrix.html) | 主要総説8件の文献マトリクス。4社の取り上げ度と原著（DOI）へのリンク付き |

## 特徴

- **自己完結型**：CSSを埋め込み、外部依存なし。オフラインでもレイアウトが崩れない（文献リンクのクリックのみネット接続が必要）
- **ライト／ダーク両テーマ対応**：閲覧環境のテーマに追従
- ブラウザでHTMLファイルを直接開いて閲覧可能
